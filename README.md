# SwiftTickets

## 一、项目的基本介绍

12306 铁路购票服务是与大家生活和出行相关的关键系统，包括会员、购票、订单、支付和网关等服务。

SwiftTickets是一个仿12306的高并发铁路票务系统。

### 前端页面

```shell
npm init vite
	->y
	->[project_name]
	->Vue
	->JavaScript
cd [project_name]
npm install 
npm run dev 
```
